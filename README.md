## Description

Create an application that can be used to write, save, and delete notes. This application uses an Express backend and saves and retrieves note data from a JSON file.

---

## User Story

AS A user  
I WANT to be able to write and save notes  
I WANT to be able to delete notes I've written before  
SO THAT I can organize my thoughts and keep track of tasks I need to complete

---

## Usage

This is a note-taking application featuring:
- **Create Notes:** Write new notes with title and text content
- **Save Notes:** Persist notes to JSON file storage
- **View Notes:** Display all saved notes in a sidebar list
- **Delete Notes:** Remove notes that are no longer needed
- **Express Backend:** RESTful API for all CRUD operations
- **Responsive Interface:** Clean, user-friendly design

### Features:
- [x] Write and save notes
- [x] Delete previously written notes
- [x] View list of all saved notes
- [x] Click saved notes to view content
- [x] Persistent data storage using JSON file

## Installation

To run this project locally:
```bash
# Clone the repository
git clone [repository-url]

# Install dependencies
npm install

# Run the application
npm start
```

**Note:** This project is no longer deployed but can be run locally for demonstration purposes.
